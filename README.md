# LANCopyPaste

### a simple tool for copy text on local netwrok devices

### Use `8000` Port:

```
python3 main.py
```

### run on custom port:
```
python3 main.py -p 2020
```

### set output path to save text file
```
python3 main.py -p 2020 -o /tmp/clipboard.txt
```
